Introduction

JavaScript is a high-level programming language used primarily for web development.

It allows developers to create dynamic and interactive web applications.

JavaScript runs in the browser as well as on servers using environments like Node.js.

console.log("Hello, World!");
Variables and Data Types

Variables store data values that can be reused later.

JavaScript supports several data types like numbers, strings, and booleans.

Variables can be declared using var, let, or const.

let age = 20; const name = "Suraj";
  • Number
  • String
  • Boolean
Functions

Functions are reusable blocks of code.

They help organize logic and reduce repetition.

function greet() { return "Hello"; }
  • Function declaration
  • Function expression
Control Flow

Control flow determines how code executes.

Conditional statements make decisions.

if (x > 5) { console.log(x); }
  • if / else
  • switch
Arrays and Objects

Arrays store ordered collections of data.

Objects store data as key-value pairs.

let arr = [1, 2, 3]; let user = { name: "Alex", age: 25 };
  • Arrays use indexes
  • Objects use keys
